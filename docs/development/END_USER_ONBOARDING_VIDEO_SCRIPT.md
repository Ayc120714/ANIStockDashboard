# End-User Onboarding Video Script

Purpose: record one standard onboarding video for Premium users based on current website navigation.

Target length: 8-10 minutes
Audience: new end users with Premium full access
Format: screen recording + voiceover

## 1) Intro (0:00-0:30)

Voiceover:
"Welcome to ANI Stock Dashboard. In this onboarding video, we will walk through the complete navigation and daily workflow from Profile setup to market modules."

On screen:
- Open `Profile -> Onboarding` tab
- Show page title

## 2) Profile and account basics (0:30-1:30)

Voiceover:
"Start from Profile. In Account Details, verify your name and email, add your mobile number if needed, and keep your password updated."

On screen:
- Open `Profile -> Account Details`
- Show mobile save flow
- Show `Change Password` button

## 3) Broker integration (1:30-3:00)

Voiceover:
"Next, configure broker connectivity in Broker Integration. Select your broker, enter required credentials, save, and validate session. This is required for live portfolio and order data."

On screen:
- Open `Profile -> Broker Integration`
- Select one broker
- Show `Save`, `Validate & Create Session`, and status chips

## 4) Start of daily analysis: Overview (3:00-4:00)

Voiceover:
"Daily workflow starts with Overview. Use this page to read market breadth, index direction, and sector-level movement."

On screen:
- Open `Overview`
- Scroll key sections

## 5) Opportunity discovery: Screens and Advisor (4:00-5:10)

Voiceover:
"After understanding market context, move to Screens to filter opportunities and Advisor for guided signal interpretation."

On screen:
- Open `Screens`
- Open `Advisor`

## 6) Risk and execution monitoring (5:10-6:20)

Voiceover:
"Use Portfolio Manager to track open positions and risk. Use Alerts to monitor key symbols and react quickly."

On screen:
- Open `Portfolio Manager`
- Open `Alerts`

## 7) Derivatives and macro context (6:20-7:20)

Voiceover:
"For broader positioning, use F&O, Commodities, and Forex. These modules help align derivatives and global context with equity decisions."

On screen:
- Open `F&O`
- Open `Commodities`
- Open `Forex`

## 8) Events and close (7:20-8:00)

Voiceover:
"Use Events for announcements and important updates. Repeat the workflow daily: Overview, Screens/Advisor, Portfolio/Alerts, then derivatives context."

On screen:
- Open `Events`
- Return to `Profile -> Onboarding`

## Recording Checklist

- Resolution: 1920x1080
- Zoom: 100%
- Cursor: visible and slow movement
- Hide all secrets in credential fields
- Keep narration speed around 125-140 words per minute
- Export: MP4 (H.264), 1080p

## Suggested Video Title

"ANI Stock Dashboard Premium Onboarding - Full Navigation Walkthrough"
